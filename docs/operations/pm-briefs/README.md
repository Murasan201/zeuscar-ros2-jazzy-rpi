# PMブリーフディレクトリ

## 概要

PMへのエスカレーション記録を管理する。

## 命名規則

`<BacklogID>.md`

例: `STORY-001.md`

## ステータス定義

- **Open**: 新規打ち上げ
- **Follow-up**: PM対応中
- **Resolved**: 解決済み

## 必須セクション

1. 背景・課題
2. 求める判断・リソース
3. 希望期限
4. ステータス
5. 対応ログ
